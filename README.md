## CPP00

This 42-school project is an introduction to object-oriented programming. It introduces the notion of class, instance, constructor, etc. 

## Concepts

- Class and instances
- Member attributes and non member
- Constructor and destructor
- Iomanips
- Namespaces
- Visibility
- This
